# Contributing

Thanks for your interest in improving this list.

Please keep these rules in mind:

- Add only high-quality, maintained projects with clear documentation.
- Keep entries scoped to the OpenCLI ecosystem.
- Write descriptions as short, objective phrases ending with a period.
- Keep list ordering and grouping consistent.
- Include only one new entry per pull request unless entries are tightly related.
- Run `awesome-lint` and fix any reported issues before opening a PR.

When adding a plugin, include:

- Project name as a markdown link.
- A concise description.
- The main repository URL.
- A short reason the entry is useful, when not obvious.
